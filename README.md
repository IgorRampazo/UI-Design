# UI-Design

> Repositório onde armazeno interfaces e conceitos de UI criados para **prática, estudo próprio e experimentação visual**.

> Repository where I store UI interfaces and concepts created for **practice, personal study, and visual experimentation**.

<br>

## Sobre/About

> Este repositório reúne layouts, telas e ideias de interface desenvolvidas durante meus estudos em design de interfaces e front-end. O foco é explorar **hierarquia visual, cores, tipografia e usabilidade**, sem compromisso com projetos comerciais.

> This repository brings together layouts, screens, and interface ideas developed during my studies in interface and front-end design. The focus is on exploring **visual hierarchy, core elements, typography, and usability**, without any commitment to commercial projects.
